```

```

```

```

```

```

## Formating the text
 
```

```

```
**bold**
```

```

```

## Blockquote code


```

```

## Lists

**Bullets**

```
+ One bullet
+ Second Bullet
+ Third bullet
```

**Ordered lists**

```
1. First item in list
2. Second item in list
```

**Item list**

```
- First item
- Second item
- Third item
```

## Horizontal rule

```

```

## Links


```

```

# veil

## Installation

Repository: [https://github.com/Veil-Framework/Veil/](https://github.com/Veil-Framework/Veil/)


Quick install for kali:

```bash
apt update
apt install -y veil
/usr/share/veil/config/setup.sh --force --silent
```


To run veil:

```bash
veil
```

# rar - Winrar

## Installation

```shell-session
 wget https://www.rarlab.com/rar/rarlinux-x64-612.tar.gz

tar -xzvf rarlinux-x64-612.tar.gz && cd rar

```


## Basic use

```shell-session
rar a ~/test.rar -p ~/test.
```
